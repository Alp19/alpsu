# alpsu
ALP System Updater

It is used to update macOS, Linux and BSD based operating systems.

Using:

<pre>
git clone https://github.com/Alp19/alpsu.git
</pre>
<pre>
cd alpsu 
</pre>
<pre>
bash alpsu.sh or ./alpsu.sh 
</pre>
